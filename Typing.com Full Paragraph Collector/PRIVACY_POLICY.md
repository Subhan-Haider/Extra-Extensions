# Privacy Policy for Typing.com Full Paragraph Collector

**Effective Date: February 5, 2026**

At **Typing.com Full Paragraph Collector**, we take your privacy seriously. This Privacy Policy explains how our browser extension handles user data.

## 1. Data Collection & Usage
**Typing.com Full Paragraph Collector** is a focused utility tool.
*   **No Personal Data Collection**: We do not collect, store, or transmit any personally identifiable information (PII), such as usernames, passwords, or typing speeds.
*   **Local Extraction**: The extension reads the text currently visible on your Typing.com lesson page to reconstruct the paragraph. This data is only used to display the text within the extension popup.
*   **No Tracking**: We do not use cookies or analytics to track your activity.

## 2. Permissions & Justification
To provide its core functionality, the extension requires the following permissions:
*   `activeTab`: Allows the extension to interact with the Typing.com page only when you explicitly click the extension icon.
*   `scripting`: Required to safely read the text elements from the webpage and display them to you.

## 3. Data Storage
This extension does not store any browsing data long-term. Extracted text is held in temporary memory and is cleared when you close the extension popup.

## 4. Third-Party Services
This extension does not communicate with any external APIs or third-party servers.

## 5. Security
All code is open-source and can be reviewed. We do not use remote code execution.

## 6. Contact
If you have questions about our privacy practices, please open an issue on our [GitHub Repository](https://github.com/Subhan-Haider/Extra-Extensions).
