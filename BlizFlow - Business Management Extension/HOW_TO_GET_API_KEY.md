# How to Get Your BlizFlow API Key

## Quick Steps

1. **Log in to BlizFlow**
   - Go to [https://blizflow.online](https://blizflow.online)
   - Sign in with your account credentials

2. **Navigate to Settings**
   - Click on your profile icon (top right corner)
   - Select **Settings** from the dropdown menu

3. **Go to Security Section**
   - In the Settings page, find and click on **Security** in the left sidebar
   - Or look for the Security tab

4. **Access API Keys**
   - In the Security section, find **API Keys** or **API Management**
   - Click on it to open the API Keys page

5. **Create or Copy API Key**
   - If you don't have an API key:
     - Click **Create New API Key** button
     - Give it a name (e.g., "Chrome Extension")
     - Click **Create**
   - If you already have an API key:
     - Find your existing key in the list
     - Click **Show** or **Copy** to reveal the key
     - ⚠️ **Important**: Copy it immediately - you may only see it once!

6. **Paste in Extension**
   - Open the extension options page
   - Paste the API key in the "API Key" field
   - Click **Save API Settings**
   - Click **Test Connection** to verify it works

## API Key Format

BlizFlow API keys typically start with:
- `sk_` (Secret Key)
- `bf_` (BlizFlow Key)
- Or a long alphanumeric string

## Security Tips

1. **Never share your API key** publicly
2. **Don't commit API keys** to version control (Git)
3. **Use different keys** for different applications
4. **Revoke keys** if you suspect they're compromised
5. **Rotate keys** periodically for better security

## Troubleshooting

### "API key not found"
- Make sure you copied the entire key
- Check for extra spaces before/after the key
- Try creating a new API key

### "Connection failed"
- Verify your internet connection
- Check if the API URL is correct
- Make sure your account is active
- Contact support if the issue persists

### "Invalid API key"
- The key might have been revoked
- Create a new API key
- Make sure you're using the correct key format

## What the API Key Does

The API key allows the extension to:
- ✅ Sync your invoices, clients, and expenses
- ✅ Access your business data securely
- ✅ Create new records (invoices, clients, etc.)
- ✅ Update existing records
- ✅ Get real-time statistics
