## Permission justification
**tabs**: Required to access and manipulate browser tabs.
**storage**: Required to save user preferences and extension state locally.
**contextMenus**: Required to add items to the right-click menu for quick access to extension features.
**activeTab**: Required to access the current tab's content only when the user explicitly invokes the extension.
**unlimitedStorage**: Required to store unlimited amount of data locally.
**tabGroups**: Required for core functionality: tabGroups
**scripting**: Required to inject scripts or styles into pages.
