# Browser Extensions Collection

A collection of browser extensions for Chrome/Edge, each serving a unique purpose to enhance your browsing experience.

## Extensions Included

1. **BlizFlow - Business Management Extension** - Business management and productivity tools
2. **Chronos Visual Context Manager** - Visual context management
3. **ClipTrack - Clipboard History Manager** - Clipboard history tracking and management
4. **CodeLens Extension Source Explorer** - Source code exploration and analysis
5. **GhostType AI** - AI-powered typing assistance
6. **Humanize AI** - Make AI text sound more human
7. **Image Converter Pro** - Professional image conversion tool
8. **LootOps Epic & Steam Games** - Game deals and loot tracking
9. **Offline Video Downloader** - Download videos for offline viewing
10. **Simple Ad Blocker** - Block common ads on websites
11. **Smart Paste** - Intelligent clipboard pasting
12. **Typing.com Full Paragraph Collector** - Collect and manage typing paragraphs

## Repository Structure

Each extension is contained in its own directory with complete documentation:

```
extension-name/
├── manifest.json                      # Extension manifest
├── LICENSE                            # MIT License
├── README.md                          # User documentation
├── SECURITY.md                        # Security policy
├── CHROME_STORE_LISTING.md           # Store submission guide
├── CHROME_STORE_PRIVACY_FORM.md      # Privacy compliance form
├── PRIVACY_COMPLIANCE.md             # Compliance information
└── [extension files]                  # Source code and assets
```

## Installation

### From Chrome Web Store
Each extension can be installed from the Chrome Web Store (links to be added after publication).

### Manual Installation (Developer Mode)
1. Download or clone this repository
2. Open Chrome/Edge and navigate to `chrome://extensions/` or `edge://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked"
5. Select the specific extension directory you want to install
