## ğŸ—ï¸ Architecture & Privacy Manifesto
*Why Subhan Extensions are built differently.*

1. **Zero-Knowledge Principle**: We do not have a user database. Your data is stored in the local browser context and never leaves your machine.
2. **Manifest V3 Standard**: All tools are built using the latest Chrome Extension API to ensure maximum speed and compatibility.
3. **No External Dependecies**: Where possible, we avoid remote fonts and scripts to prevent supply-chain vulnerabilities.

## â“ Frequently Asked Questions

**Q: Are these extensions really data-private?**  
A: Yes. 90% of our tools work entirely offline. For the 10% that use AI APIs (like Humanize), we use encrypted HTTPS channels and do not store your data on our end.

**Q: Do I need to pay for anything?**  
A: The source code is 100% free under the MIT license. Some AI features may require your own API keys for high-volume use.

**Q: Will these work on Firefox or Safari?**  
A: We prioritize Chromium-based browsers (Chrome, Edge, Brave, Opera). Safari and Firefox support is not currently in the roadmap.
