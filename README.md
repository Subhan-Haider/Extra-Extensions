## ğŸ—ï¸ Architecture & Privacy Manifesto
*Why Subhan Extensions are built differently.*

1. **Zero-Knowledge Principle**: We do not have a user database. Your data is stored in the local browser context and never leaves your machine.
2. **Manifest V3 Standard**: All tools are built using the latest Chrome Extension API to ensure maximum speed and compatibility with future browser updates.
3. **No External Dependecies**: Where possible, we avoid remote fonts and scripts to prevent supply-chain vulnerabilities and improve load times.
4. **Local processing**: Whether it's image conversion or typing simulation, the heavy lifting happens on your CPUâ€”not a remote server.
