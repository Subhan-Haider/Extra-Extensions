# Auto Email Follow-up

A premium browser extension built for high-performance and utility.

## Features
- Modern UI: Glassmorphic, responsive design.
- Privacy First: All data is stored locally using chrome.storage.local.
- Manifest V3: Compliant with the latest Chrome extension standards.
- Performance: Lightweight and fast.

## Installation
1. Download or clone this repository.
2. Go to chrome://extensions/ in your browser.
3. Enable Developer mode.
4. Click Load unpacked and select the Auto Email Follow-up folder.
