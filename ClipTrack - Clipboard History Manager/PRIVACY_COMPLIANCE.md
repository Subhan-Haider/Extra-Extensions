## Permission justification
**storage**: Required to save user preferences and extension state locally.
**clipboardRead**: Required to read clipboard data for pasting into the extension.
**clipboardWrite**: Required to copy data to the clipboard.
**activeTab**: Required to access the current tab's content only when the user explicitly invokes the extension.
**downloads**: Required to download files generated or selected by the extension.
